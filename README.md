# Portfólio Pessoal

Este repositório contém o código do meu portfólio online, desenvolvido para apresentar meus projetos, habilidades e experiências na área de desenvolvimento web.  
O site foi construído com **HTML, CSS e JavaScript**, focando em um design limpo, responsivo e fácil de navegar.

## Funcionalidades
- Seção de projetos com cards interativos
- Layout responsivo para diferentes dispositivos
- Links diretos para visualizar ou acessar cada projeto
- Estrutura simples e organizada para fácil manutenção

## Objetivo
O portfólio serve como uma vitrine dos meus trabalhos e como ponto de contato profissional, permitindo que recrutadores e colaboradores conheçam minhas habilidades e projetos.

## Estrutura
- `index.html` → Página principal do portfólio
- `style.css` → Estilos e layout do site
- `script.js` → Funcionalidades interativas
